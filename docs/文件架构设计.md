> 类图

**此处考虑迁移**
1. 使用code/path关联
2. 文件路径拆分为path+storageId

```mermaid
classDiagram

    %% 文件标识
    class FileMask {
        <<uint8>>
        +const PermissionRead : FileMask
        +const PermissionWrite : FileMask
        +const PermissionExec : FileMask
        +const PermissionDelete : FileMask
        +const PermissionShare : FileMask
        +const PermissionModify : FileMask

        +CanRead() bool
        +CanWrite() bool
        +CanExec() bool
        +CanDelete() bool
        +CanShare() bool
        +CanModify() bool

        +AddRead() void
        +AddWrite() void
        +AddExec() void
        +AddDelete() void
        +AddShare() void
        +AddModify() void

        +RemoveRead() void
        +RemoveWrite() void
        +RemoveExec() void
        +RemoveDelete() void
        +RemoveShare() void
        +RemoveModify() void
    }
    %% 元信息
    class FileModule {
        +string path 路径
        +string storageId 存储
        +FileMask fileMask 文件权限
        +string user 用户
        +string group 权限
        ...

    }
    FileModule *-- FileMask : contains
    %% 路径下加密配置
    class PathEncryption  {
        +string path 无存储前缀
        +string storageId 存储
        +string encryptionCode 关联的加密方式
        +GetByPath(storageId,path string) Encryption 获取加密方式
    }
    %% 加密配置
    class Encryption{
        +string code 唯一编码
        +string type 加密方式
        +map[string]string 加密凭据
        +GetByCode(code string) Encryption 获取单个加密配置
        +Export() List<Encryption> 导出
        +Import() void 导入
    }
    %% 存储
    class Storage {
        ...
        +string id
        ...
    }
    FileModule "1" --o "0..1" PathEncryption
    PathEncryption "n" --o "1" Encryption
    FileModule "n" --o "1" Storage
```
-> 参考示例：[FileMash.go](https://gist.github.com/dezhishen/4a61dc2ee8d4f2ef69eab1be1027fde4#file-filemask-go)

> 加/解密流程


```mermaid
graph LR
A([开始])
A-->B{路径下是否加/解密}
B--Y-->C(调用加/解密模块)
B--N-->D(存储是否加/解密)
D--Y-->C
C-->C1{是否有指定方式的加/解密的适配器}
C1--Y-->C2(调用适配器进行加/解密)
C1--N-->C3(报错)-->F
C2-->E(插件处理数据)
D--N-->E
E-->F([结束])
```

# OpenList 文件系统架构

## 文件系统代码
```mermaid
graph LR
F0[layers/file] --> F1
```


## 文件配置详情

```mermaid
graph LR
A[权限配置]
    A --> B(权限组A)
    A --> C(权限组A)
    A --> D(.......)
    B --> B1(是否加密内容: bool)
    B --> B2(是否加密名称: bool)
    B --> B3(是否隐藏目录: bool)
    B --> B4(是否压缩文件: bool)
    B --> B5(是否分卷文件: bool)
    B --> B6(文件权限设置: 10bits)
    B --> B7(文件所有用户: number)
    B --> B8(文件分卷大小: number)
    B --> B9(单独访问密码: string)
    B --> BA(文件加密密码: string)

```
1、路径加密每一个路径不能重叠，添加的时候需要遍历扫描
2、如果加密文件内容，必须输入密码，只加密文件名不需要
3、需要有个默认权限：默认不加密且公开，但是可以修改它

> 文件配置页面

```mermaid
graph LR
A[路径配置]
    A --> B(/path1/) --> B1(权限组A)
    A --> C(/path2/) --> C1(权限组A)
    A --> D(/path3/pic.jpg) --> D1(权限组B)
    A --> E(/path4/file.mp4) --> E1(权限组B)
    A --> F(...............)
```

> 加密执行流程

```mermaid
graph LR
A(上传/path2/file.mp4)
    A --> B1(匹配/path1/) --> C1(未命中)
    A --> B2(匹配/path2/) --> C2(命中) --> D1(执行加密)
    F(file.mp4) --> G1(加密文件) --> J1(file.mp4.u8BMxMpD.enc)
    F(file.mp4) --> G2(加密名称) --> J2(ZmlsZS5tcDQ=.b64)
    F(file.mp4) --> G3(压缩分卷) --> J3(file.mp4.f001.zip)
    X1[ZmlsZS5tcDQ=.b64] -->X4[.zip] -->X2[.f001] -->X3[.u8BMxMpD.enc]
    Y1[加密文件名部分] --> Y4[压缩部分] --> Y2[分卷] --> Y3[加密部分]
```
压缩加密后缀：ZEC，只加密文件的：ENC，只压缩的后缀ZIP
PS：如果都没有命中，当然是不加密/压缩直接上传原始文件


> 文件访问流程

```mermaid
graph LR
A1(/path2/file.mp4.enc)
    A1 --> B1(后缀是ENC) --> C1(显示为file.mp4)
    B1 --> B11(匹配/path1/) --> C11(未命中)
    B1 --> B12(匹配/path2/) --> C12(命中) --> D11(解密)

A2(/path2/ZmlsZS5tcDQ=.b64.u8BMxMpD.enc)
    A2 --> B2(后缀是ENC) --> C2(解密文件名) -->D2(显示为file.mp4)
    B2 --> B21(匹配/path1/) --> C21(未命中)
    B2 --> B22(匹配/path2/) --> C22(命中) --> D21(解密)
```

PS：如果匹配不上并且文件加密，需要输入密码
